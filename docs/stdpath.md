# VIM
`vim.fn.stdpath({what})`
| what | example | description |
| :--- | :--- | :--- |
| config | $HOME/.config/nvim | Neovim configuration directory |
| data | $HOME/.local/share/nvim | Neovim data directory. e.g. plugins, LSP, Treesitter |
| state | $HOME/.local/state/nvim | Neovim state directory. e.g. cache log, session, undo file |
| cache | $HOME/.cache/nvim | Neovim cache directory. e.g. temporary files |
| log | $HOME/.local/log/nvim | Neovim log directory. e.g. log file |
| runtime | $HOME/.local/share/nvim/runtime | Neovim runtime directory |
